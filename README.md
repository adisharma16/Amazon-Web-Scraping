# Web Scraping Project 2.0

A Python-based web scraping project that collects product information from Amazon using `BeautifulSoup`, `Requests`, and `LXML`.

## Features

* Scrapes product details from Amazon
* Uses custom headers and user-agent handling
* Parses HTML content using BeautifulSoup
* Lightweight and beginner-friendly project
* Built in Jupyter Notebook (`.ipynb`)

## Technologies Used

* Python
* BeautifulSoup4
* Requests
* LXML
* CSV
* Jupyter Notebook

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/web-scraping-project.git
```

Move into the project folder:

```bash
cd web-scraping-project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the Jupyter Notebook:

```bash
jupyter notebook
```

Then open:

```bash
web scraping 2.0.ipynb
```
